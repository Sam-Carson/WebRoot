Web Resources

Mozilla

Mozilla is one of the most informing resources to anything web development. Mozilla offers tutorial information for HTML5, CSS3, JavaScript, HTTP's, API's and DOM. Not only is Mozilla a leading brand in web development, they also have their own web browser, Firefox Developer Edition, that is caterred to developers. The browser is great for beginners who are learning HTML, CSS and JavaScript. The brower is loaded with features that will make you're time learning, less of a headache and more productive.

Rating: 4 out of 5

CSS-Tricks

CSS-Tricks is the one stop shop for finding anything CSS related. You can certainly find resources on HTML and JavaScript sprinkled in, but the focus is CSS. What makes this resource stand out is the Almanac feature. The Almanac contains detailed information of every selector and property. Have you ever watched a tutorial and wondered what one of the CSS properties is and how it works? Head over to CSS-tricks for all the information you'll need to learn. Also, check out the Guides to find tutorials on everything CSS related.

Rating: 5 out of 5

dribbble

Sometimes all you need to get started is some inspiration. Whether you want to push your skills to new heights or you need to create a unique design for a client, dribbble is the place to go for inspiration. Find anything from Web Design, Mobile Applications, UI/UX, and much more.

Rating: 4.5 out of 5

w3schools

w3schools has been my go to site for quick references and help when whenever I have questions about HTML, CSS or JavaScript. What makes w3 so useful for this course is the easy access to responsive web design that is featured under the HTML and CSS section. Something that sets w3schools apart form other resourceful material is there How to section. Look for anything you may need help with and you can very likely find it there.

Rating: 5 out of 5

DesignCourse

DesignCourse is a Youtube channel created by a fullstack developer named Gary. Nowadays, you can learn just about anything on Youtube and because of the complexity of software development, having a video to watch and follow along can be much more convenient than reading documentation. I'm personally a visual learner so this makes things much easier for. I've followed along with a few of these videos in the past and I can vouch for Gary, he puts out some pretty neat stuff. His videos will help beginners develop their HTML, CSS and JavaScript skills.

Rating: 4.5 out of 5
